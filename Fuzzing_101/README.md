## Setup

Spawn a Docker instance:

```shell
docker run -ti -v $HOME:/home aflplusplus/aflplusplus
```

Or, consider make it an alias and add it to `~/.zshrc` or `~/.bashrc`:

```shell
alias afl="docker run -ti -v $HOME:/home aflplusplus/aflplusplus"
```
